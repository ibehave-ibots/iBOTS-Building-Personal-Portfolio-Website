# iBOTS Workshop: Building Personal Portfolio Website

**Date**: Jan 15th-16th, 2025
**Instructor**: Dr. Sangeetha Nandakumar

## Installation 

**STEP 1: Clone this repository**

```
git clone https://github.com/ibehave-ibots/iBOTS-Building-Personal-Portfolio-Website.git
cd iBOTS-Building-Personal-Portfolio-Website
```

**STEP 2: Install the environment**

```
pixi shell
```

**STEP 3: Test Hugo Availability**
```
hugo version
go --v
```

**STEP 4 (Optional): Register Bash Kernel**
```
python -m bash_kernel.install
```
